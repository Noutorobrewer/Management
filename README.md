# Management

マネジメントスキル向上のためのイシュー管理。

## 目的

- 自分自身のコーチング（セルフリフレクション）
- 部下のコーチング（1on1・成長支援）
- 課題の発見と改善のPDCLAサイクル
- 一流のマネージャーを目指す

## Issue構成

| カテゴリ | ラベル | 内容 |
|---------|--------|------|
| 自己コーチング | `self` | 自分の課題・内省・行動変容 |
| 1on1 | `1on1` | 1on1の質向上・フレームワーク |
| 部下育成 | `coaching` | 部下の成長支援・フィードバック |
| チーム | `team` | チーム運営・文化づくり |
| 学習 | `learning` | 書籍・理論・フレームワークの学び |

## 使い方

- 各Issueのコメントに日次・週次の記録を蓄積
- 月1回の振り返りでIssueを更新・クローズ・新規作成
